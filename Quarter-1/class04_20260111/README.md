### Fundamental Prompting Techniques

1. **Zero-Shot Prompting:** Direct questions without examples
2. **One-Shot Prompting:** Single example to guide format
3. **Few-Shot Prompting:** Multiple examples to establish patterns
4. **System Prompting:** Setting overall context and behavior
5. **Role Prompting:** Assigning specific expertise or character
6. **Contextual Prompting:** Providing relevant background information
